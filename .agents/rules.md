# Regra de Auditoria com Codex CLI (Planos de Implementação)

- Sempre que criar ou atualizar um plano de implementação (`implementation_plan.md`), você deve executar a auditoria do Codex CLI sobre as alterações planejadas.
- Utilize o comando de revisão adequado (`codex review` ou `codex exec`) para auditar o plano.
- O plano só deve ser submetido para aprovação do usuário após a validação e verificação de eventuais apontamentos feitos pelo Codex CLI.
